# Convert JSON to XLSX

This is a simple Node.js application that converts JSON data to an Excel file (XLSX).

## Installation

1. **Clone the repository:**

```bash
git clone https://github.com/your-username/convert-json-to-xlsx.git
```

2. **Install dependencies:**
   Navigate to the project directory and run:

```bash
yarn install
```

3. **Usage:**
   Prepare your JSON data:
   Ensure that your JSON data is correctly formatted. You can modify the data.json file in the project directory or provide your own JSON file.

Run the script:

```bash
yarn start [path_to_your_json_file.json]
```

Replace `[path_to_your_json_file.json]` with the path to your JSON file. For example:

```bash
yarn start D:\my-input.json
```

4. **Check the output:**
   After running the script, a new Excel file named `xxx-excel.xlsx` will be generated in the project directory.
